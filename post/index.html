March 10, 2021

Accessible and Tagged PDF

Accessible PDF A PDF document is accessible to me if I can open and read it. That might depend on the circumstances and the document. It is a human-related property of the document. A real human being reading the document. Tagged PDF A PDF document is tagged if roughly speaking from it an equivalent HTML document can be obtained, by applying some technically specified software. The real human being reading the document is not present.
